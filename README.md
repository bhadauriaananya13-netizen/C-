A console-based Supermart Management System built using C language to manage product details, quantities, prices, and company-wise product lists.

DESCRIPTION

A simple console-based Supermart Management System developed using C programming.

This project helps manage product information in a supermart by storing product name, company name, available quantity, and price.

## Features

- Add product information
- Display all products
- List products by company
- Count total products available
- Exit system

## Technologies Used

- C Programming
- Structures
- Arrays
- Loops
- Conditional Statements
- String Handling
